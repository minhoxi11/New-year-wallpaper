# Year Progress Wallpaper

Auto-updating wallpaper for Samsung S22+ showing days passed in the year.

## Setup

1. Create a new GitHub repository
2. Push this folder to the repository
3. Enable GitHub Actions (Settings → Actions → Allow all actions)
4. The wallpaper will be generated automatically every day at midnight UTC

## KLWP Setup

1. Open KLWP on your phone
2. Add a **Shape** → Rectangle (full screen)
3. Go to **FX** → **Texture** → **Bitmap**
4. Set Type: **File/Web Img**
5. Set URL to: `https://raw.githubusercontent.com/YOUR_USERNAME/year-progress-wallpaper/main/wallpaper.png`
6. Save and apply

The wallpaper will refresh automatically when KLWP reloads.

## Manual Trigger

Go to Actions tab → Generate Wallpaper → Run workflow
